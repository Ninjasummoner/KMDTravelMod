This license is written for KMD Travel. It is intended to let players, server owners, modpack makers, and addon developers use and build around KMD Travel while preventing unofficial rehosts, renamed copies, or standalone redistributed forks of the mod.

This is not an OSI-approved open-source license. KMD Travel is source-available and addon-friendly, but redistribution of KMD Travel itself is limited by the terms below.

## 1. Definitions

**KMD Travel** means the source code, compiled jars, models, textures, sounds, generated documentation, branding, names, logos, and other files in this repository unless a file clearly states otherwise.

**Official distribution** means a release published by Ninjasummoner or by a project page controlled by Ninjasummoner.

**Addon** means a separate mod, datapack, script, resource pack, compatibility layer, integration, or other project that depends on KMD Travel but does not include substantial KMD Travel source code or assets.

**Modpack** means a curated Minecraft pack that includes the official KMD Travel jar as a dependency or included mod.

**Substantial code or assets** means enough KMD Travel code, models, textures, UI assets, sounds, or branding that the new project can reasonably replace KMD Travel or appear to be KMD Travel under another name.

## 2. What You May Do

You may:

- View, read, and study the source code.
- Use KMD Travel in personal worlds and private servers.
- Include official KMD Travel releases in public or private modpacks.
- Create and publish addons that require KMD Travel as a dependency.
- Create datapacks, configs, recipes, event profiles, translations, compatibility files, and resource packs for KMD Travel.
- Fork the repository privately for testing, learning, bug investigation, or preparing contributions.
- Submit issues, pull requests, patches, translations, documentation fixes, or compatibility improvements.
- Reference KMD Travel APIs, registries, item IDs, block IDs, commands, config formats, and documented behavior when building addons.

## 3. What You May Not Do Without Written Permission

You may not:

- Rehost, mirror, or redistribute KMD Travel jars outside approved modpack distribution.
- Publish modified KMD Travel jars as your own release.
- Rename, rebrand, or reskin KMD Travel and distribute it as a separate mod.
- Copy substantial KMD Travel source code into another mod or project.
- Copy substantial KMD Travel assets into another mod, resource pack, or project.
- Remove or replace KMD Travel branding in order to present the mod as your own work.
- Sell KMD Travel, access to KMD Travel, or modified KMD Travel builds.
- Use KMD Travel's name, logo, screenshots, textures, models, or branding in a way that suggests your project is official unless it is approved by Ninjasummoner.
- Create an addon that bundles KMD Travel internally instead of requiring the official KMD Travel mod as a dependency.

## 4. Addon Rules

Addons are allowed and encouraged when they follow these rules:

- The addon must be a separate project.
- The addon must require KMD Travel as a dependency if it needs KMD Travel features.
- The addon must not include KMD Travel source code or assets beyond small references needed for compatibility.
- The addon may use KMD Travel IDs, commands, config formats, event profile formats, and public behavior to integrate with the mod.
- The addon may say it is compatible with KMD Travel or requires KMD Travel.
- The addon must not imply that it is an official KMD Travel release unless approved by Ninjasummoner.

Good examples:

- A mod that adds new KMD event profiles and mobs while depending on KMD Travel.
- A datapack that changes KMD recipes.
- A resource pack that changes KMD textures for personal or pack use.
- A quest-pack integration that runs KMD commands.
- A compatibility mod that connects KMD Travel to another travel or quest system.

Not allowed without permission:

- A fork that republishes KMD Travel as a competing standalone mod.
- A mod that copies KMD Travel's map, travel post system, event editor, and assets so KMD Travel is no longer required.
- A reuploaded KMD jar on another site.

## 5. Modpack Permission

You may include official KMD Travel releases in public or private modpacks without asking first, as long as:

- The modpack uses an official KMD Travel jar.
- The modpack does not claim ownership of KMD Travel.
- The modpack does not modify and redistribute KMD Travel itself unless permission is granted.

Private forks are allowed for testing, learning, bug fixing, and preparing pull requests.

Public forks are allowed for contribution and review purposes, but they may not be used as public replacement distributions of KMD Travel. If you want to maintain a public fork for a special platform, loader, Minecraft version, or compatibility target, ask Ninjasummoner first.
