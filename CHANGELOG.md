# Changelog

All notable changes to KMD Travel will be tracked here.


## 0.2.0

- Added custom event profiles with in-game editing.
- Added configurable passive and aggressive travel events.
- Added custom event mobs, command actions, delays, dimensions, biomes, event weights, and avoid chance.
- Added shared travel posts and shared-post crafting control.
- Added parchment-style fast travel map, discovered-area panel, pins, labels, zooming, panning, and map caching.
- Added world/server-specific event profile storage.
- Added support work for Forge 1.20.1, NeoForge 1.21.x, and Fabric 1.21.x builds.
