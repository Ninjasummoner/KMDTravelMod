# Security Policy

KMD Travel is a Minecraft mod and does not process sensitive account credentials.

If you find a crash, server exploit, unsafe command behavior, or data-loss issue, please open a GitHub issue with reproduction steps and logs. If the issue should not be public, contact Ninjasummoner privately before posting details.

Please include:

- Loader and Minecraft version
- KMD Travel version
- Server or singleplayer
- Minimal reproduction steps
- Latest log or crash report
